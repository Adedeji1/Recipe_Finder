# Recipe Finder Application

A responsive and interactive Recipe Finder web application that allows users to search for meals, explore recipe details, filter results, and save favorite recipes for later. The app uses a public API and focuses on clean JavaScript, good UX, and modern CSS layouts.

## Features

* Search for recipes using a public API (TheMealDB)
* Display results in a responsive CSS Grid layout
* View recipe details in a modal
* Filter recipes by category and ingredients
* Save favorite recipes using `localStorage`
* Form validation with user feedback
* Loading indicators and error handling
* Fully responsive design (mobile, tablet, desktop)


## Technologies Used

* **HTML5** – Semantic markup
* **CSS3** – Flexbox, CSS Grid, responsive layouts
* **JavaScript (ES6+)** – API handling, DOM manipulation, state management
* **Public API** – TheMealDB
* **LocalStorage** – Persisting favorite recipes


## How to Run Locally

1. Clone the repository

   ```bash
   git clone https://github.com/your-username/recipe-finder.git
   ```

2. Navigate into the project folder

   ```bash
   cd recipe-finder
   ```

3. Open `index.html` in your browser

*No additional setup required.*
